Тестовый стенд Qual IT:
Для запуска стенда создайте папку "Working Project" на диске C, скопировать .jar файл в папку, открыть консоль cmd.exe,
переключиться в папку с .jar файлом командой "cd C:\Working Project", ввести в консоли команду "java -jar standForPractics.jar" для запуска стенда.
Стенд доступен по адресу http://localhost:8080.
Для дальнейшей работы со стендом будет использоваться вкладка "Песочница" - "Товары".
Функционал вкладки: просмотр товаров в списке и добавление новых товаров.

Требования:
Страница Список товаров: содержит таблицу с товарами с полями "Наименование", "Тип", "Экзотический" и кнопку "Добавить"
Кнопка "Добавить": цвет кнопки синий, при нажатии: открытие окна "Добавление товаров".
Окно Добавление товаров содержит: Наименование, Тип, Экзотический, Сохранить.
Поле "Наименование": тип поле ввода, длина без ограничений и принимает любой символ
Поле "Тип": тип выпадающий список, на выбор варианты "Фрукт", "Овощ"
Поле "Экзотический": тип чекбокс, варианты true или false.
Кнопка "Сохранить" : при нажатии: добавляется новый товар с введенными данными, в БД добавляется новая строка записи.
Цель тестирования:
Проверка функционала добавления товаров и валидация отображения товаров в списке.
